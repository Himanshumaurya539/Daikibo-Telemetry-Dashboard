# Daikibo-Telemetry-Dashboard
This project focuses on analyzing telemetry data from Daikibo factories to identify machine downtime patterns and improve operational efficiency.  Using data visualization and analytics tools, I created an interactive dashboard to monitor and analyze downtime across factories and device types.

---

## 🎯 Objectives
- Analyze telemetry data from industrial machines  
- Identify downtime trends and patterns  
- Compare performance across factories and device types  
- Build an interactive dashboard for decision-making  

---

## 🛠️ Tools & Technologies
- Tableau (Data Visualization)
- Microsoft Excel (Data Preparation)
- Data Analytics Techniques  

---

## 📊 Dashboard Features

### 🔹 Down Time per Factory
- Identifies factories with highest downtime  
- Helps prioritize maintenance efforts  

### 🔹 Down Time per Device Type
- Shows which machine types are most problematic  
- Enables targeted improvements  

### 🔹 Downtime Trend Over Time
- Tracks downtime patterns across time  
- Helps detect spikes and anomalies  

### 🔹 Interactive Filters
- Select a factory to filter all visuals  
- Dynamic dashboard for better insights  

---

## 📈 Key Insights
- Certain factories contribute significantly more downtime  
- Specific device types are more prone to failure  
- Downtime patterns vary over time, indicating possible operational inefficiencies
